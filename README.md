# HQ Wealth Wise — Redesign Concept

An unofficial concept mockup exploring a redesign of [hqwealthwise.com](https://hqwealthwise.com), built to show the site owner a possible direction.

Live preview: https://wealthwise-demo.locyfish.com (temporary demo hosting, not the production domain)

## Pages
- `index.html` — Home
- `about.html` — About
- `podcast.html` — Podcast
- `contact.html` — Contact
- `style.css` — Shared styles

## Notes
- Content in dashed "placeholder slot" boxes needs real input from the site owner (bio, photos, real contact info, episode links).
- The contact form is intentionally disabled in this mockup (visual only, no backend wired up yet).
